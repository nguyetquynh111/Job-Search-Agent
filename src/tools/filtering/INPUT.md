# Input

| Field | Type | Meaning |
| --- | --- | --- |
| `jobs` | list of `Job` | Jobs to filter. |
| `preferences` | `CandidatePreferences` | Candidate's required preferences. |

A `Job` includes ID, title, company, location, remote status, description, requirements, salary, and source.

`CandidatePreferences` includes target titles, locations, remote preference, job types, minimum salary, and excluded keywords.
