# Resume Tailoring Tool
This tool creates a job-specific resume from the original resume, fit analysis, and approved evidence.

- See [INPUT.md](INPUT.md) for input fields.
- See [OUTPUT.md](OUTPUT.md) for output fields.
- Contract: `TailorResumeInput` to `TailorResumeOutput`.

The future implementation must keep the original resume unchanged and must not add unsupported facts.
